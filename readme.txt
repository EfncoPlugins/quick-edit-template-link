Displays the current template in the admin bar with a link directly to the template in theme editor (if the current user has access to it)

== Description ==

This plugin adds a handy link at the top of the admin bar when you're viewing your website.

It shows you what template is currently being used.

If you have access to edit templates it links to the theme editor with the current template selected for editing


== Installation ==

There are no settings to configure, just install, activate and go to the front end of your website

== Frequently Asked Questions ==

= Can Editors and Subscribers view the template? =
They can see the name of the template in the adminbar, but they cannot click the link into the theme editor unless you have given them capabilities
